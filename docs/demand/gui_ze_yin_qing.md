# 规则引擎

规则引擎让用户对设备作更精细化的设置, 例如在以下场景，主动邮件通知用户
1. 当设备的某个物模型变量超出或低于正常区间.
2. 希望设备在固定时间，或每隔一段时间完成其它操作时(路灯每天晚8点到早6点亮)
