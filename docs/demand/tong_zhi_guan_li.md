# 通知管理 

通知管理允许用户设置不同方式的通知，例如邮件，微信，钉钉等, 目前系统只支持邮件通知方式。  
在设置通知配置前，需要先设置一个通知模板。

#### 新建模板  

点击页面通知管理-> 通知模板, 点击新建通知模板, 新建一个通知模板.
